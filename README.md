# Retail Sales Analytics Dashboard

## Project Overview
This Tableau dashboard analyzes retail sales performance across different product categories.  
The project focuses on sales trends, promotion impact, product placement performance, and revenue comparison.

---

## Tools Used
- Tableau
- SQL
- PostgreSQL

---

## Dashboard Features
- Sales Performance Analysis
- Product Category Comparison
- Promotion Impact Analysis
- Revenue Difference Analysis
- Product Placement Heatmap
- Interactive Category Selector

---

## Key Insights
- Jackets generated the highest revenue.
- Promotions affected category performance differently.
- Product placement had a strong impact on sales volume.
- Sweaters showed high average sales per product.
